MIDI Captain MAX — Firmware Installation
========================================

Prerequisites:
  - CircuitPython 7.x installed on your device
  - Device connected via USB and mounted as CIRCUITPY or MIDICAPTAIN

Recommended: Use the deploy script
------------------------------------------------

  macOS/Linux:

    First install (includes CircuitPython libraries):

      ./deploy.sh --install

    Update existing firmware:

      ./deploy.sh

    Options:
      --install     Full install with CircuitPython libraries
      --eject       Eject device after deploy (forces clean reload)
      --fresh       Overwrite config.json with default
      --libs-only   Only install CircuitPython libraries

  Windows (PowerShell):

    First install (includes CircuitPython libraries):

      .\deploy.ps1 -Install

    Update existing firmware:

      .\deploy.ps1

    Options:
      -Install      Full install with CircuitPython libraries
      -Eject        Eject device after deploy (forces clean reload)
      -Fresh        Overwrite config.json with default
      -LibsOnly     Only install CircuitPython libraries
      -MountPoint   Specify drive letter (e.g. -MountPoint E:\)

  Both scripts auto-detect your device type and preserve your
  existing config.json (button mappings, colors, etc.).

Manual Installation (if deploy scripts are unavailable)
------------------------------------------------

  1. Connect your MIDI Captain via USB.
  2. Hold Button 1 (upper-left switch) while powering on.
  3. Copy ALL files and folders to the device drive,
     replacing existing files.
  4. The device restarts automatically after the copy completes.
     If not, unplug and replug USB.

  Important: If you have a custom config.json with your own
  button mappings, keep your existing one — don't overwrite it.

Recovery
------------------------------------------------

  If the device ends up in a bad state, it's fully recoverable:
  mount the device again, erase the contents, and copy the
  firmware files again.

Files included:
  deploy.sh        Installation script (macOS/Linux)
  deploy.ps1       Installation script (Windows PowerShell)
  code.py          Main firmware entry point
  boot.py          Startup configuration
  config.json      Default config (STD10)
  config-mini6.json  Default config (Mini6)
  core/            Firmware modules
  devices/         Hardware definitions
  fonts/           Display fonts
  lib/             CircuitPython libraries
  VERSION          Firmware version identifier
