# Emulator Directory

This directory contains scripts and configurations for testing MIDI Captain firmware using the **rp2040js-circuitpython** emulator.

## Quick Start

```bash
# 1. Setup emulator (one-time)
./emulator/setup.sh

# 2. Run automated tests
./emulator/test.sh

# 3. Run interactively
./emulator/run.sh
```

## What's Here

- **setup.sh** - Downloads and configures the emulator (run once)
- **run.sh** - Run firmware interactively in emulator
- **test.sh** - Run automated tests and verify output
- **configs/** - Test configuration files
- **fs/** - Firmware filesystem (generated by setup.sh)
- **rp2040js-circuitpython/** - Emulator clone (generated by setup.sh)
- **circuitpython-7.3.1.uf2** - CircuitPython firmware (downloaded by setup.sh)

## Documentation

## Gitignore

The following are automatically generated and excluded from git:

- `fs/` - Firmware filesystem copy
- `rp2040js-circuitpython/` - Emulator clone
- `*.uf2` - CircuitPython firmware
- `*.log` - Test logs
